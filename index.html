How it works

Three steps to prevent disasters

Pattern matching, not AI. Zero false positives.

Step 1

Export traces

Export from your observability platform or capture directly from your AI agent.

# LangSmith export langsmith export > trace.jsonl # Or capture directly python agent.py > trace.jsonl
Step 2

Run analysis

AgentCorrect scans for payment issues, SQL disasters, and destructive commands.

# Run locally agentcorrect analyze trace.jsonl # In CI/CD pipeline agentcorrect analyze || exit 1
Step 3

Block on failures

Exit code 2 fails your CI/CD pipeline. Fix issues before they reach production.

# GitHub Actions - run: | python run_agent.py > trace.jsonl agentcorrect analyze trace.jsonl
