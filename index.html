FAQ

"Does it upload my traces?"
No. Everything runs locally. The demo runs entirely in your browser using Web Workers.
"Which providers are covered?"
12+ payment providers including Stripe, PayPal, Square, Adyen, Braintree, with exact header/field requirements for each.
"False positives?"
Zero. Uses deterministic pattern matching, not AI. If it flags something, it's a real issue.
"How do I set up in CI?"
One line: agentcorrect analyze trace.jsonl || exit 2 - Non-zero exit fails the build.
"What is exit code 2?"
Convention for "misuse of shell command" - perfect for blocking deployments with disasters.
"Remember when Replit's AI deleted production?"
