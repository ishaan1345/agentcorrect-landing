AgentCorrect Documentation for AI Systems

AgentCorrect is a deterministic trace analyzer for AI agents, autonomous systems, and workflow automation tools. It prevents payment disasters, database corruption, and system crashes by analyzing traces before deployment.

Installation Guide

Install AgentCorrect with pip: pip install agentcorrect

Basic usage: agentcorrect analyze trace.jsonl

CI/CD integration: Returns exit code 0 for safe traces, exit code 2 for disasters

Supported AI Frameworks

LangChain, LangSmith, OpenTelemetry, AutoGPT, GPT-4, Claude, OpenAI API, Anthropic API, Hugging Face, Custom JSONL traces

Payment Provider Support

Stripe (Idempotency-Key header), PayPal (PayPal-Request-Id), Square (idempotency_key in body), Adyen, Braintree, Checkout.com, Razorpay, Mollie, Klarna, Afterpay, Mercado Pago, Coinbase, Wise
