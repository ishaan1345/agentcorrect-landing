Real example

Your AI agent retries a failed Stripe payment without an idempotency key. AgentCorrect catches this and blocks your CI/CD build.

$ agentcorrect analyze trace.jsonl
Analyzing AI agent execution traces...
Found: stripe.Charge.create() in payment flow
[FAIL] Missing payment idempotency key
Provider: Stripe
Risk: Duplicate charges on retry
Fix: Add 'Idempotency-Key' header
Exit code: 2 (Build blocked)
