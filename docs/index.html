Payment APIs
Missing Idempotency
AI doesn't know that payment APIs need idempotency keys. Without them, network retries cause duplicate charges.
stripe.charges.create() // No Idempotency-Key
SQL Operations
Unbounded Deletes
AI doesn't understand that DELETE without WHERE is catastrophic. Or that WHERE 1=1 matches everything.
DELETE FROM users WHERE 1=1
Cache Commands
Redis FLUSHALL
AI doesn't realize FLUSHALL wipes the entire cache, causing immediate production outages.
redis.flushall()
