# AgentCorrect Landing Page

Production-grade Next.js 14 landing page for AgentCorrect.

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm run start
```

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Performance

- CLS < 0.02
- LCP < 1.8s on 4G
- Lighthouse scores: Performance ≥ 90, Accessibility ≥ 95

## Security

- No external trackers
- No cookies
- Offline-first design
- CSP headers configured
